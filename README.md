# AULAGIT
## Iniciando Projeto GIT
- Repositório Local
- Repositório Remoto
- git config --global user.name "(nome)"
- git config ==global user.email "(email)"
-git init
- git remote add origin (link repositório)
## Fazer Projeto
- Faz umas coisas '-'
##Stage
-git add .
-git commit -m "(mensagem do commit)"
##Push
-git push origin (master/main)
